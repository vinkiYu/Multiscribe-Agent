## 7. 风险、遗留与取舍

- **取舍**：curate Prompt 不再看到 URL、source、category；这是任务包明确要求，因为最终链接和来源通过 `id -> UnifiedData` 回映，且精选阶段只需要标题和摘要。若未来排序依赖来源或分类，需要单独扩展投影契约。
- **风险**：150 字符是字符数上限，不是 Token 上限；CJK、Emoji 和 HTML 的 Token 成本仍由 P30 的 token counter 和上下文预算闭环兜底。
- **风险**：当前测试以 JSON 字符长度作为精简比例口径，不等同于真实 tokenizer Token 比例；任务包验收明确采用该口径，模型窗口效果仍应在 P30.2 或评估包中用真实 tokenizer 复核。
- **未做的事**：没有改 `CURATE_PROMPT` 文本、没有改变 overview/fanout 投影、没有增加来源/分类过滤逻辑，也没有进行真实 LLM 或 Webhook e2e。
