## 详细任务

### T1. `domain/models.py` — 核心领域模型

全部用 Pydantic v2（`BaseModel`，`model_config = ConfigDict(frozen=False)`，按需）。必须定义：

1. **UnifiedData**（统一数据母线）：
   - 字段：`id, title, url, description: str`，`published_date: str`，`ingestion_date: str | None`，`source, category: str`，`author: str | None`，`status: str | None`，`metadata: dict[str, Any] = {}`（扩展点，可含 tags/ai_summary/ai_score/ai_score_reason）。
   - 验证：`published_date` 非空；`id` 非空。

2. **AIMessage / AIResponse**：
   - `AIMessage`：`role: Literal["user","assistant","system","tool"]`，`content: str`，`tool_calls: list[ToolCall] | None`，`tool_call_id: str | None`，`name: str | None`。
   - `ToolCall`：`id, name: str`，`arguments: dict[str, Any] | str`（字符串时运行期 JSON.parse）。
   - `AIResponse`：`content: str`，`tool_calls: list[ToolCall]`，`usage: TokenUsage | None`，`raw: dict | None`。
   - `TokenUsage`：`input_tokens, output_tokens, total_tokens: int`。

3. **AgentDefinition**：`id, name, description, system_prompt: str`，`provider_id, model: str`，`temperature: float = 0.7`，`tool_ids: list[str] = []`，`skill_ids: list[str] = []`，`mcp_server_ids: list[str] = []`，`streaming: bool = False`，`is_hidden: bool = False`，`category: str | None = None`。

4. **WorkflowDefinition**：`id, name, description: str`，`steps: list[WorkflowStep]`。
   - `WorkflowStep`：`id, name: str`，`step_type: Literal["agent","workflow"]`，`agent_id: str | None`，`workflow_id: str | None`，`input_map: dict[str, str] | None`，`next_step_id: str | None`，`next_step_ids: list[str] | None`，`enabled: bool = True`，`config: dict[str, Any] = {}`。
   - **Loop 扩展字段**（P10/P11 用，P1 先定义）：`max_iterations: int | None`，`exit_condition: str | None`。

5. **ToolDefinition**：`id, name, description: str`，`parameters: dict[str, Any]`（JSON Schema），`is_builtin: bool = False`。

6. **MCPConfig**：`id, name: str`，`transport: Literal["stdio","sse","streamable_http"]`，`command: str | None`，`args: list[str] = []`，`url: str | None`，`headers: dict[str,str] = {}`，`env: dict[str,str] = {}`，`enabled: bool = True`。

7. **SkillEntry / SkillFrontmatter**：`SkillFrontmatter(name, description: str; bins: list[str] = [])`；`SkillEntry(id, name, description, instructions: str; dir_path: str | None; is_builtin: bool; frontmatter: SkillFrontmatter; files: list[str] = [])`。

8. **SourceData**（DB 实体）：`id, title, url, description: str`，`published_date, source, category: str`，`author: str | None`，`metadata: dict`，`fetched_at, ingestion_date: str`，`adapter_name: str`，`status: str | None`。

9. **TaskLog**：`id: str | None`，`task_id, task_name: str`，`start_time, end_time: str | None`，`duration_ms: int | None`，`status: Literal["running","success","error","interrupted"]`，`progress: float | None`，`message: str | None`，`result_count: int | None`。

10. **CommitRecord**：`date, platform, file_path, commit_message, commit_time, full_content: str`。

11. **ScheduleTask**：`id, name: str`，`task_type: Literal["full_ingestion","adapter","agent_summary","agent_deal","daily_digest"]`，`cron: str`，`enabled: bool = True`，`config: dict[str, Any] = {}`，`last_run: str | None`，`last_status: str | None`，`last_error: str | None`。

12. **MemoryEntry / KBCategory / KBDocument / KBChunk**：先定义结构占位（P16/P17 细化），P1 给基础字段即可。

每个模型配 docstring。

### T2. `domain/ports.py` — 仓储接口（Protocol）

定义 Protocol（不实现），供 infra 实现、上层 mock：
- `KvRepository`：`get(key) -> Any | None`，`set(key, value, ttl_seconds=None)`，`delete(key)`。
- `EntityJsonRepository`：泛型思路——`get(table, id) -> dict | None`，`save(table, id, data)`，`list_all(table) -> list[dict]`，`delete(table, id)`。
- `SourceDataRepository`：`save_batch(items: list[UnifiedData], adapter_name) -> int`，`query(filters) -> list[SourceData]`，`search_fts(query, limit) -> list[SourceData]`，`get_by_date_range(start, end) -> list[SourceData]`。
- `TaskLogRepository`：`create(log) -> str`，`update(log_id, **fields)`，`get(log_id) -> TaskLog | None`。
- `ApiKeyRepository`：定义 P2 用到的方法签名（create/get_by_prefix/update_status 等）。

用 `typing.Protocol`，方法标 `async`（infra 用 aiosqlite 实现时即异步；测试可用同步 fake）。

### T3. `core/errors.py` — 领域异常

```python
class MultiscribeError(Exception): """所有领域异常基类。"""
class ConfigError(MultiscribeError): ...
class ValidationError(MultiscribeError): ...
class ProviderError(MultiscribeError): ...
class ToolExecutionError(MultiscribeError): ...
class WorkflowError(MultiscribeError): ...
class PublisherError(MultiscribeError): ...
class AdapterError(MultiscribeError): ...
class AuthError(MultiscribeError): ...
```

### T4. `config.py` — Settings（pydantic-settings）

- `class ProviderConfig(BaseModel)`：`id, name: str`，`type: Literal["openai","anthropic","google","ollama"]`，`api_key: str = ""`，`base_url: str = ""`，`use_proxy: bool = False`。
- `class AdapterConfig / PublisherConfig / StorageConfig`：`id, type: str`，`enabled: bool = True`，`config: dict[str, Any] = {}`。
- `class SystemSettings(BaseSettings)`（`model_config = SettingsConfigDict(env_prefix="MULTISCRIBE_", env_file=".env", extra="allow")`）：
  - 字段：`system_password: str = ""`，`jwt_secret: str = ""`，`db_path: str = "data/database.sqlite"`，`log_level: str = "INFO"`，`active_ai_provider_id: str = ""`，`http_proxy: str = ""`。
  - `ai_providers: list[ProviderConfig] = []`（默认预置 4 个 provider 占位）。
  - `adapters: list[AdapterConfig]`、`publishers: list[PublisherConfig]`、`storages: list[StorageConfig]`（默认预置 RSS / feishu_bot+wecom_bot+github+rss / r2+github_storage）。
  - `closed_plugins: list[str] = []`。
  - `selection_fetch_days: int = 2`，`selection_query_field: str = "ingestion_date"`。
- 提供 `class ConfigService`（单例风格函数 `get_settings() -> SystemSettings`）：实现三层合并 `default_settings → .env/SystemSettings → kv 覆盖`。kv 覆盖部分 P1 先留接口（`async def load_overrides() -> dict` 返回 {}），P2 接 DB 后实现。
- 顶层提供 `DEFAULT_SETTINGS: SystemSettings` 常量（含预置 provider/adapters/publishers）。
