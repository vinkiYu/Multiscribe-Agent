## 详细任务

### T1. `renderers/feishu_card.py` — 卡片渲染器

把「精选摘要列表」渲染为飞书交互卡片（interactive card）JSON。参考飞书开放平台「自定义机器人 - 消息卡片」规范。

```python
def render_digest_card(title: str, items: list[DigestItem], *, footer: str | None = None) -> dict:
    """返回飞书 interactive 卡片 JSON。

    卡片结构：
    - header: 标题（如 "📰 每日精选 · 2026-07-15"）
    - body: 多个 div/markdown 模块，每条精选一个模块（标题加粗 + 摘要 + 来源链接）
    - footer: 可选统计信息
    """
```

- `DigestItem`：`title: str`, `summary: str`, `url: str`, `source: str`, `score: float | None`（P11 产出，P7 先定义 dataclass）。
- 卡片用飞书 `i18n_elements` / `elements` 结构，markdown 模块支持加粗/链接/列表。
- 提供纯文本降级 `render_digest_text(title, items) -> str`（webhook 不支持卡片时用）。

### T2. `plugins/builtin/publishers/feishu.py` — FeishuPublisher

```python
class FeishuPublisher(BasePublisher):
    metadata = PluginMetadata(
        id="feishu_bot", type=PluginType.PUBLISHER, name="飞书机器人",
        description="通过自定义机器人 Webhook 推送到飞书群",
        icon="feishu",
        config_fields=[
            ConfigField(key="webhook", label="Webhook URL", type="url", required=True, scope="adapter"),
            ConfigField(key="secret", label="签名密钥", type="password", required=False, scope="adapter"),
        ],
    )
    async def publish(self, content, options=None) -> dict:
        """content 可以是卡片 dict 或文本 str；options 含 webhook/secret。"""
```

核心逻辑：
1. 取 `options.webhook`（或 content 里带）。
2. **签名**（若配 secret）：飞书签名算法
   - `timestamp = str(int(time.time()))`
   - `string_to_sign = f"{timestamp}\n{secret}"`
   - `hmac_sha256(string_to_sign, "")` → base64 → `sign`
   - payload 加 `"timestamp": timestamp, "sign": sign`。
3. POST webhook（httpx，超时 10s）。
4. 飞书返回 `{"StatusCode":0,"StatusMessage":"success"}` 或 `code:0` → 成功；否则抛 `PublisherError`。
5. 重试：失败指数退避（最多 3 次，间隔 1s/2s/4s）。
6. 返回 `{status, response}`。

**签名实现要点**（飞书官方算法）：
```python
import hmac, hashlib, base64, time
def gen_sign(timestamp: int, secret: str) -> str:
    string_to_sign = f"{timestamp}\n{secret}"
    hmac_code = hmac.new(string_to_sign.encode("utf-8"), digestmod=hashlib.sha256).digest()
    return base64.b64encode(hmac_code).decode("utf-8")
```
