## 详细任务

### T1. `pyproject.toml`（用 uv 管理，src layout）

要求：
- `[build-system]` 用 `hatchling`（uv 默认友好）。
- `[project]`：name=`multiscribe-agent`，version=`0.1.0`，requires-python=`>=3.12`，description，readme=`README.md`，license GPL-3.0。
- `[project.scripts]`：`multiscribe-agent = "multiscribe_agent.cli:main"`。
- `[project.dependencies]`（MVP 基线依赖，按后续包逐步增，P0 先放最小集）：
  - `pydantic>=2.7`，`pydantic-settings>=2.3`
  - `structlog>=24.1`
  - `click>=8.1`（CLI）
- `[project.optional-dependencies]`：
  - `dev`：`pytest>=8`，`pytest-asyncio>=0.23`，`respx>=0.21`（httpx mock），`ruff>=0.5`，`mypy>=1.10`，`pre-commit`，`types-click`
- `[tool.hatch.build.targets.wheel]`：`packages = ["src/multiscribe_agent"]`
- `[tool.ruff]`：target-version=`py312`，line-length=`100`；select=`["E","F","W","I","UP","B","C4","SIM","ANN","S","PT","RUF"]`；per-file-ignore 测试放宽 ANN/S。
- `[tool.ruff.format]`：默认。
- `[tool.mypy]`：strict=true，packages=`["multiscribe_agent"]`，plugins=`["pydantic.mypy"]`。
- `[tool.pytest.ini_options]`：asyncio_mode=`auto`，testpaths=`["tests"]`，markers=`["e2e: 真实外部依赖测试，默认跳过"]`，addopts=`-m "not e2e"`。

## 验收条件

1. `uv sync`（含 `--extra dev`）成功，无依赖冲突。
2. `uv run python -m multiscribe_agent --version` 输出 `multiscribe-agent 0.1.0`。
3. `uv run pytest -q` 通过（3 个冒烟测试全绿）。
4. `uv run ruff check .` 无错误。
5. `uv run ruff format --check .` 无需改动。
6. `uv run mypy src` 无错误。
7. `pre-commit run --all-files` 全绿（可本地验证）。
8. `pyproject.toml` 配置项齐全（ruff/mypy/pytest 段落都在）。
